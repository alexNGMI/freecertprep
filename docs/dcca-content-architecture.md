# Schneider DCCA Content Architecture

## Source Basis

Primary sources:

- Schneider Electric University overview: Schneider describes its Data Center Certified Associate certification as foundational knowledge of data center critical physical infrastructure, including cooling, power, racks, cabling, fire protection, management, and physical security.
- Schneider Electric University DCCA Exam Development Path: the official learning path lists the preparation modules used as the content map for this bank.

The first implementation follows the Schneider course sequence rather than inventing a vendor-neutral outline. Because Schneider does not publish public domain percentages for DCCA, the practice pool uses near-even weighting across the official course areas.

## Exam Model

- Cert id: `schneider-dcca`
- Display name: Schneider Data Center Certified Associate
- Code: `DCCA`
- Provider: Schneider Electric
- Practice pool: 750 questions
- Simulator: 100 questions / 120 minutes / 70% practice pass target
- Question types: single-choice, multiple-response, matching, ordering

## Quality Bar

DCCA questions should test field judgment, not trivia alone:

- Data center technician scenarios: rack-and-stack work, maintenance planning, capacity review, post-incident review, deployment handoff.
- Physical infrastructure vocabulary: UPS, generator, PDU, A/B feeds, N+1, 2N, containment, blanking panels, pre-action fire suppression, structured cabling, DCIM.
- Practical decisions: what to inspect, what risk is present, which design principle applies, what order a technician should follow.
- Explanations must state why the correct answer is right and why the distractors are unsafe, incomplete, or unrelated.
